# Bank Marketing Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Bank Marketing Dataset using Python.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed

- Dataset Overview
- Data Cleaning
- Missing Value Analysis
- Statistical Summary
- Customer Age Analysis
- Job Distribution
- Marital Status Analysis
- Education Analysis
- Housing Loan Analysis
- Personal Loan Analysis
- Contact Method Analysis
- Campaign Analysis
- Term Deposit Subscription Analysis

## Visualizations

- Bar Charts
- Pie Charts
- Histograms
- Stacked Bar Charts

## Dataset

Bank Marketing Dataset (UCI Machine Learning Repository)
